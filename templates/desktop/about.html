About {{ session['site_name'] }}

This is a hobby project made by a hobby programmer. It came about after a brief discussion about how nice it would be if there was a personal link aggregator like Reddit.

After looking around at several solutions already created (and one stupid email) I decided to create a simple (and badly coded) page for users to post their internet finds.

It doesn't have many features, but it has enough for what I need. This was inspired by not only Reddit but sites like Hacker News and Lobsters.
